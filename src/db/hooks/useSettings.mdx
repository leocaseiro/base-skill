# useSettings

Reactive hook that subscribes to the anonymous profile's settings document in RxDB.

## Return value

Returns `{ settings, update }`. The `settings` object merges defaults with the RxDB document when present.

```ts
type SettingsDoc = {
  id: string;
  profileId: string;
  volume: number;
  speechRate: number;
  ttsEnabled: boolean;
  showSubtitles: boolean;
  updatedAt: string;
};
```
